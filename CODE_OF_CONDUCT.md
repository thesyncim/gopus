# Code of Conduct

## Our Standard

`gopus` aims to be a respectful, technically rigorous project where people can collaborate without being talked down to.

Examples of behavior that help:

- Being respectful in disagreement
- Focusing feedback on code, behavior, and evidence
- Welcoming contributors with different backgrounds and experience levels
- Sharing context, reproductions, and references when making technical claims
- Accepting correction gracefully

## Unacceptable Behavior

This project does not allow:

- Harassment, intimidation, threats, or stalking
- Discriminatory or demeaning language
- Personal attacks, insults, or deliberate humiliation
- Trolling, bad-faith disruption, or repeated refusal to engage constructively
- Publishing private information without permission
- Sexualized language or unwelcome sexual attention

## Scope

This code of conduct applies in project spaces, including issues, pull requests, review comments, and other public interactions tied to `gopus`.

## Reporting

To report a conduct issue, contact `thesyncim@gmail.com`. Please include links, screenshots, or other context if you have them.

## Enforcement

Maintainers may edit or remove content, pause conversations, close threads, reject contributions, or block participants whose behavior violates this code of conduct.

When possible, enforcement will explain what behavior needs to change. Repeated or severe violations may lead to longer-term restrictions.
